# Football-match-analysis-sql-data-warehouse
SQL-based Football Match Analysis Data Warehouse using Star Schema Design. Includes player performance analytics, team statistics, and stadium insights.

# Project Overview
Star Schema football analytics database built using MySQL.
Analyzes player performance, team stats, and stadium insights.

## Tables
- Player_Dim
- Team_Dim
- Date_Dim
- Stadium_Dim
- Match_Fact

## Analysis Included
- Total Goals per Team
- Top Assist Providers
- Average Possession
- Player Performance
- Goals Per Match Efficiency
